Course Syllabus

1) Course Basics

Course: ESE 124 — Programming Fundamentals (4 credits)

Coordinator: Dr. Alex Doboli

Description:
The course presents fundamental and more advanced C programming concepts...

Prerequisite: Major or AOI

Course Outcomes:
• An ability to develop algorithmic solutions for fundamental computing problems
• An ability to develop C programs for fundamental algorithms
• An ability to debug, test, and validate C programs
• An ability to effectively document C programs

2) Textbooks & Major Topics

Textbook:
1. zybooks: "Programming in C" (Link)
2. A. Hilton, A. Bracy, "All of Programming," 2015, ISBN: 9780996718202

Major Topics Covered:

  • Demo C program for simple calculator
  • Basic types, operators, bitwise ops
  • Conditionals (if, switch), loops (for, while, do)
  • Arrays, strings, pointers, pointer arithmetic
  • Structured types (struct, union), type definitions (typedef)
  • Abstract data types (*.h, *.c)
  • Dynamic memory allocation
  • Linked lists, hash tables
  • FSM implementation
3) Labs, Projects, and Grading

Laboratory & Design Projects:
- Weeks 1-2: Bitwise operations
- Weeks 3-4: Loops & conditionals (scanner)
- Weeks 5-6: Number sorting
- Weeks 7-8: Stack ADT
- Weeks 9-10: Table-based FSM
- Weeks 11-12: Linked lists, hash table
- Week 13: Completing the course project

Grading:
Midterm exam (25%), Final exam (25%), weekly homework (15%), Lab assignments (20%), Course project (15%), plus up to 5% extra credit based on in-class participation.

Grading Policy:
- Must score ≥ 45 points in final exam (out of 100) to pass the course.
- A: 90—100, A-: 85—89.99, B+: 80—84.99, B: 75—79.99, B-: 70—74.99,
C+: 65—69.99, C: 60—64.99, C-: 55—59.99, D+: 52—54.99, D: 48—51.99, F: 0—47.99.
